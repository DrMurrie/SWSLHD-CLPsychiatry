This contains tools to use within SWSLHD Macarthur CL Psychiatry including referral forms, guides, and questionnaires. 

[Referral Template Form](cl-psych-referral.html)

This form generates a referral for you to place into the EMR referral form (Liaison Psychiatry Consult). Using this template will ensure that your consult is prioritied.

[Hospital Anxiety and Depression Scale (HADS)](hads-scale.html)

This form is a good questionnaire for patients in hospital that screens for anxiety and depressive symptoms. This form generates a full report that you can copy into your note.
